Was ich gerade über HTML und CSS lerne

Tags

Es gibt verschiedene Arten von Tags. Was sie aber alle (im generellen wenigstens) gemeinsam haben ist, dass sie mit diesen < und > Zeichen markiert sind.
Ein Opening Tag wird wie folgend konstruiert <...>. Die Closing Tags werden genauso konstruiert, haben aber ein Slash nach dem kleiner Zeichen: <./...>.
Im Folgenden liste ich die verschieden Arten von Tags und wozu sie dienen auf.
Ich musste nach dem Opening Tag ein Punkt setzten, damit man den Tag tatsächlich sehen konnte. Ignoriert einfach die Punkte :).

Überschriften

Um Überschriften zu kennzeichen benutzt man innerhalb der Klammern h1 bis h6. H1 ist dabei die grösste Überschrift, die anderen sind dann Unter-unter...Überschriften.

Absätze

Vom englischem paragraph benutzt man für die Absätze innerhalb des Tags ein kleines p. <.p> ... <./p>

Zeilen

Um Zeilenbrüche zu definieren benutzt man den Tag <.br> am Ende einer Zeile. So kannst du Entscheiden ob die Sätze in einer Zeile oder in mehreren erscheinen.
Z.B:
Hallo!
Wie geht es dir?
Oder
Hallo! Wie geht es dir?

Bilder

Für Bilder, muss man diese zuerst in einem Ordner mit der Website speichern, danch den Tag <.img src="Name der Datei" /> benutzen.
Dies kann man auch in einem neuen Absatz machen.

Link auf andere Websiten

Um ein Link zu anderen Websiten einzufügen benutzt man den folgenden Tag: <.a href="Link"> Name der Website oder ähnliches <./a>.
So kann man beim Draufdrücken auf eine andere Website gelangen.

Kennzeichnung von Inhaltsabschnitten

Um zu kennzeichnen, dass ein neuer Sinnesabschnitt beginnt, kann man den Tag <.hr> benutzen. Bei mir zum Beispiel erstellt der Browser dabei eine Linie unter jeder Einheit.

Druckschrift

Um Texte oder einzelne Wörter in Druckschrift zu schreiben benutzt man den Tag <.b> ... <./b>.

Kursivschrift

Wenn man wissenschaftliche Begriffe benutzt, schreib man diese normalerweise in kursiver Schrift .
Um das zu schaffen benutzt man den Tag: <.i> ... <.i/>.

Markieren

Für das Markieren von Wörtern/Texten benutzt man den Tag <.mark> ... <./mark>.

Wählen

Wenn man eine Wahl hinzufügen möchte, benutzt man den Folgenden Textblock:

<.label> for "Essen"> Such dir ein Essen aus: <./label>
<.select id="Essen">
<.option value="Spaghetti">Spaghetti<./option>
<.option value="Pizza">Pizza<./option>
<.option value="Salat">Salat<./option>
<.option value="Schnitzel">Schnitzel<./option>
<./select>

Listen

Wenn man Dinge auflisten will benutzt man den folgenden Textblock:
<.ul>
<.li> Scuba diving <./li>
<.li> 10 Bücher lesen <./li>
<.li> 3 neue Einträge in meinen Blog schreiben <./li>
<./ul>
Man kann durch einen h-Tag eine Überschrift darüber setzen.
